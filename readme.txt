[경북대학교 20172 시스템 프로그래밍 텀 프로젝트] 


			  	    도서관 자리 배정 시스템
				          <<A-plus팀>>

* 본 시스템은 도서관(열람실)에서 다수의 사람들에게 편리하게 자리를 배정해주는 것을 목적으로한다.

* 개발 팀은 시스템 프로그래밍 수강생들로 이루어졌으며 
컴퓨터 학부 4학년 안상영
지리학과    3학년 송재영
컴퓨터 학부 3학년 신은정
컴퓨터 학부 3학년 신승민으로 이루어져 있다.


* 주요 기능 : 시스템의 주요 기능은 자리배정, 자리반납, 시간 연장이다.

- 자리배정은 스케너를 통해 자신의 학번을 입력한 사용자에게 현재 자리를 사용중이지 않은 경우에 한하여 
  자리를 배정해주는 기능이다.

- 자리반납은 자리를 사용 중인 사용자가 자리를 반납하게끔 도와주는 기능이다.

- 시간 연장은 자리를 사용 중인 사용자의 사용시간을 연장하게끔 도와주는 기능이다.


* 사용방법

1. 자리를 배정 받기 위해선 자신의 학번이 입력되어있는 학생증을 스캐너를 통해 스캔하여 학번을 입력하거나
   키보드를 통해 직접 학번을 입력해도 된다.

2. 학번을 입력한 후 터치스크린에 나와있는 좌석들 중 하늘 색 좌석 가운데 하나를 터치하거나 마우스를 이용
   해 클릭하면 좌석이 배정이 되며 화면에 해당 좌석이 빨간색으로 바뀐다.

3. 자리를 반납하기 위해선 자신의 학번을 다시 입력해야한다. 자신이 좌석을 이미 사용 중이라면 해당 좌석을
   반납할 것인지 시간을 연장할 것인지 선택하라는 표시가 나온다. 이 때 키보드로 2번을 입력하면 좌석을 반
   납할 수 있다.
	3-1. 만약 반납, 연장 메세지가 뜨지 않는다면 좌석이 배정되지 않은 것이다.
	3-2. 1번이나 2번이 아닌 버튼을 입력하거나 화면을 터치, 클릭하면 아무 반응도 일어나지 않는다.

4. 좌석은 일정시간이 지나면 자동으로 반납된다. 이 시간은 시스템 내부에서 설정 가능하다. 

5. 시간이 지나 자동으로 반납되기 전 사용 시간 연장을 하고 싶으면 좌석 반납과 같이 스캐너나 키보드를 통해 
   학번을 입력한다. 그리고 연장할 것인지 반납할 것인지 묻는 메세지가 나오면 키보드로 1을 눌러 시간 연장 
   옵션을 선택한다.

	* 학생증 스캔을 연속으로 받을 경우 오류가 일어날 수도 있으니 주의 요망
